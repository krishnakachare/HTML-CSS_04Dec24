HTML: Hyper Text Markup Language

Shortcut for creating HTML structure : !+Enter

<!DOCTYPE html> : Representing page/document type is HTML5
<html></html> : Root element
<head></head> : Section-1 : UI Head section contains UI meta data, other required data of body section
<body></body> : Section-2 : UI body section contains all page

Html elements : HTML is tag base language
E.g1: <html></html>

<> --> open tag
</> --> close tag
html--> tagNmae

E.g2: <h1>HTML5</h2>
HTML5 --> HTML Text

Some HTML elements are self closing elements: Its not required close/End Tag
E.g: <!DOCTYPE html>,<meta>

Html nested elements: One HTML element contains inside the other HTML element 
E.g:  <html> <head></head> <body></body> </html> 

Here <html> element contains inside the 2 HTML element of <head> & <body>, Hence we can see here the HTML nested structure

<title> : Its representing the page/tab title
<h1> : Its representing heading on UI, We have <h1> to <h6> tags
